# telephone_search

Εφαρμογή αναζήτησης τηλεφώνων σε αρχεία Excel

---

## Motivation

Κατά την διάρκεια της στρατιωτικής μου θητείας, είχαμε μια υπηρεσία που λεγόταν τηλεφωνικό κέντρο. Οι τηλεφωνητές έπρεπε κάθε φορά να ψάχνουν χειροκίνητα σε μεγάλο αρχείο Excel για να βρουν το αντίστοιχο τηλέφωνο που τους ζητήθηκε ώστε να τους συνδέσουν — μια χρονοβόρα και κουραστική διαδικασία.

Έφτιαξα αυτό το εργαλείο για να λύσω το πρόβλημα αυτό: αναζητά σε όλα τα φύλλα άμεσα και επιστρέφει αποτελέσματα σε δευτερόλεπτα.

---

## Εκδόσεις

Το project υπάρχει σε δύο εκδόσεις:

### tilSearch.py — Αρχική έκδοση
Διαβάζει απευθείας το Excel με `openpyxl` κάθε φορά που εκτελείται.

### tilSearch_sqlite.py — Έκδοση με SQLite
Κάνει import τα δεδομένα του Excel σε μια τοπική βάση δεδομένων (`phones.db`) και κάνει αναζήτηση εκεί. Το import γίνεται μόνο την πρώτη φορά ή όταν το Excel έχει αλλάξει — κάθε επόμενη εκτέλεση δεν αγγίζει καθόλου το Excel.

| | tilSearch.py | tilSearch_sqlite.py |
|---|---|---|
| Δεδομένα | Excel (ανοίγει κάθε φορά) | SQLite (φτιάχνεται μία φορά) |
| Ταχύτητα σε μεγάλα αρχεία | Αργή | Γρήγορη |
| Ενημέρωση δεδομένων | Ανοίγεις το Excel | Ανοίγεις το Excel → αυτόματο re-import |

---

## Λειτουργίες

- Αναζήτηση βάσει μονάδας, γραφείου ή τηλεφώνου
- Συνδυαστική αναζήτηση μονάδας και γραφείου
- Γενική αναζήτηση σε όλα τα φύλλα
- Πλήρης υποστήριξη ελληνικών χαρακτήρων με και χωρίς τόνους, διαλυτικά

---

## Τεχνολογίες

- Python 3
- openpyxl
- sqlite3 (ενσωματωμένο στην Python — μόνο για την έκδοση SQLite)

---

## Εγκατάσταση

```bash
pip install openpyxl
```

---

## Χρήση

**Αρχική έκδοση:**
```bash
python tilSearch.py
```

**Έκδοση SQLite:**
```bash
python tilSearch_sqlite.py
```

Στη συνέχεια επίλεξε μια από τις διαθέσιμες επιλογές αναζήτησης από το μενού.

---

## Σημειώσεις

- Για να ενημερώσεις τα τηλέφωνα, άνοιξε το αρχείο Excel και κάνε τις αλλαγές απευθείας — δεν απαιτείται καμία γνώση προγραμματισμού.
- Η αναζήτηση υποστηρίζει μερική αντιστοίχιση λέξεων. Για παράδειγμα, αναζητώντας "διευθ" θα επιστραφούν αποτελέσματα που περιέχουν "διευθυντής".
- Δεν υπάρχει πρόβλημα με τόνους και διαλυτικά. Για παράδειγμα, αναζητώντας "προισταμενος" θα επιστραφούν αποτελέσματα που περιέχουν "προϊστάμενος".
- Η αναζήτηση δεν κάνει διάκριση κεφαλαίων/πεζών — π.χ. "ΔΙΕΥΘΥΝΤΗΣ" και "διευθυντής" δίνουν το ίδιο αποτέλεσμα.
