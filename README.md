# telephone_search
Εφαρμογή αναζήτησης τηλεφώνων σε αρχεία Excel

## motivation
Κατά την διάρκεια της στρατιωτικής μου θητείας, είχαμε μια υπηρεσία που λεγόταν τηλεφωνικό κέντρο.
Οι τηλεφωνητές έπρεπε κάθε φορά να ψάχνουν χειροκίνητα σε μεγάλο αρχείο Excel για να βρουν το
αντίστοιχο τηλέφωνο που τους ζητήθηκε ώστε να τους συνδέσουν - μια χρονοβόρα και κουραστική διαδικασία.

Έφτιαξα αυτό το εργαλείο για να λύσω το πρόβλημα αυτό: αναζητά σε όλα τα φύλλα άμεσα και επιστρέφει αποτελέσματα σε δευτερόλεπτα.

## Λειτουργίες

- Αναζήτηση βάσει μονάδας, γραφείου ή τηλεφώνου
- Συνδυαστική αναζήτηση μονάδας και γραφείου
- Γενική αναζήτηση σε όλα τα φύλλα
- Πλήρης υποστήριξη ελληνικών χαρακτήρων με και χωρίς τόνους, διαλυτικά

## Τεχνολογίες

- Python 3
- openpyxl

## Εγκατάσταση

```bash
pip install openpyxl
```

## Χρήση

```bash
python tilSearch.py
```

Στη συνέχεια επίλεξε μια από τις διαθέσιμες επιλογές αναζήτησης από το μενού.

## Σημειώσεις

- Για να ενημερώσεις τα τηλέφωνα, άνοιξε το αρχείο excel και κάνε τις
  αλλαγές απευθείας — δεν απαιτείται καμία γνώση προγραμματισμού.
- Η αναζήτηση υποστηρίζει μερική αντιστοίχιση λέξεων. Για παράδειγμα,
  αναζητώντας "διευθ" θα επιστραφούν αποτελέσματα που περιέχουν "διευθυντής".
- Δεν υπάρχει πρόβλημα με τόνους και διαλυτικά. Για παράδειγμα,
  αναζητώντας "προισταμενος" θα επιστραφούν αποτελέσματα που περιέχουν "προϊστάμενος".
- Η αναζήτηση δεν κάνει διάκριση κεφαλαίων/πεζών — π.χ. "ΔΙΕΥΘΥΝΤΗΣ" και "διευθυντής" δίνουν το ίδιο αποτέλεσμα
